# ⌚ Synchronos - Boutique d'Horlogerie

**Synchronos** est un projet de site vitrine et e-commerce réalisé dans le cadre d'une collaboration pédagogique avec la section **BTS SAM (Support à l'Action Managériale)**.

---

## 📝 Contexte du projet
Ce projet simule le lancement d'une marque de montres de luxe. 
* **Ma mission :** Développer l'interface web et les fonctionnalités du site.
* **Collaboration :** Les étudiants en BTS SAM ont géré la partie commerciale, le catalogue produit et la stratégie de communication.

## ✨ Fonctionnalités
* **Catalogue dynamique :** Présentation des collections de montres.
* **Fiches produits :** Détails techniques et visuels haute définition.
* **Responsive Design :** Site entièrement consultable sur mobile et tablette.
* **Panier :** Simulation d'un parcours d'achat (selon l'état d'avancement du code).

## 🛠️ Stack Technique
* **Frontend :** HTML5, CSS3 (Flexbox/Grid), JavaScript.
* **Design :** Inspiré par les codes du luxe (épuré, typographies élégantes).
